Logo 1 — Use when

  • Navigation bar and page headers
  • Email signatures and document headers
  • Sidebar or app shell branding
  • Favicon: use mark only (inner diamond isolated)

Logo 2 — Use when

  • Hero sections and landing views
  • Print covers, internal documents
  • Presentation title slides
  • Horizontal space is limited but height is available

Dark background rules

  • Accent: #BDFF3C (full chartreuse)
  • Wordmark: #DCDCF5 (warm white)
  • Mark frame: same as wordmark
  • Never use on backgrounds lighter than #2A2A40

Light background rules

  • Accent: #8FB800 (darkened for contrast)
  • Wordmark: #0A0A12 (near black)
  • Minimum clear space: equal to mark height on all sides
  • Never recolour the accent to grey or black
